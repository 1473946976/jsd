欢迎使用 Code tool

关于插件

Code tool是一个编译器开发工具，旨在简化游戏开发流程。它提供了多种功能，包括但不限于：

  • 根据预制结构生成对应的脚本
  • 获取对Project资源的引用
  • 可定制化修改模板风格
  • 跨平台支持

无论你是独立开发者还是团队，这个插件都能帮助你提高开发效率，缩短开发时间。
